# Self Troubleshoot and System Diagnostic Tool Python
A simple Self Troubleshoot and System Diagnostic Tool (Windows only) made with Python

## Features:
-View System Information (User's name, OS, CPU, RAM)
-Performance Check (CPU, RAM, DISK Usage, Show Top Processess)
-Network Check (Check if you're connected to the Internet, IP Info, Ping Test)
-One Click Fixes (Flush DNS, Renew IP, Clear Temp Files)
-System Health Check (Estimate system's health based on CPU, RAM and DISK usage)
