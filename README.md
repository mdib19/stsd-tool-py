# Windows Self Troubleshoot and System Diagnostic Tool
A simple Self Troubleshoot and System Diagnostic Tool (Windows only) made with Python

## Features:
-View System Information (User's name, OS, CPU, RAM) <br>
-Performance Check (CPU, RAM, DISK Usage, Show Top Processess) <br>
-Network Check (Check if you're connected to the Internet, IP Info, Ping Test) <br>
-Fixes (Flush DNS, Renew IP, Clear Temp Files, SFC Scan, chkdsk, Reset Winsock/ TCP/ IP, Kill High CPU Process) <br>
-System Health Check (Estimate system's health based on CPU, RAM and DISK usage) <br>
-Event Logs Viewer <br>
-System Diagnosis <br>
